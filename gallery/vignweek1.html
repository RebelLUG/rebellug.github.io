Vignweek Contest

The Challenge

  • Build a custom vignette with the given prompt in the specified time
  • Prompts will be uploaded Monday - Saturday at 11am EST / 5pm CET
  • There will be one prompt for each weekday and one prompt for the weekend
  • Your build must be completed, posted, and submitted within 24 hours of the prompt being released on weekdays and within 48 hours for the weekend prompt
  • All rules listed below must be followed for submissions to be valid
  • First place for each prompt will receive 4 points, second place for each prompt will receive 3 points, third place for each prompt will receive 2 points, and everyone who participates for each prompt will receive 1 point
  • Prizes will be allocated to the top three builders who have the largest combined totals after the week has ended (please note you do not have to participate every day to be eligible to win

The Rules

  • Submit one (1) social media link of your build to the “Vignweek Submissions” channel in the RebelLUG Discord Server
  • Minimum size 16x16 studs and max 32x32 studs
  • Entries must be new builds, built and uploaded after the start time
  • Please use #vignweek2019 hashtag in social posts
  • No digital entries, only LEGO elements can be used, however third party blasters/minifigs (CAC, EclipseGrafix etc) are allowed
  • You may create as many entries as you like, but you may only place once
  • Submit your build before 24 hours has passed from the prompt’s release (48 hours for the last prompt). This is a firm deadline!
